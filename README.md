# 🎓 Student Life Cycle ERP QA Portfolio

## 📌 Project Overview

This repository showcases my Manual Testing portfolio for the **Student Life Cycle ERP** application. The project covers the complete student journey from admission to graduation, including multiple ERP modules, with end-to-end QA documentation.

The portfolio demonstrates industry-standard QA artifacts such as Test Plan, Test Scenarios, Test Cases, Bug Reports, Requirement Traceability Matrix (RTM), SQL Validation Report, and Test Summary Report.
